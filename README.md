## 📖 About The Project

**Online Bookstore** is a full-stack web application that allows users to browse books, register/login, and view book details. Admin users can manage the book catalog — adding, editing, or deleting books. This provides a simple real-world bookstore management system to explore and enhance. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

✔ Home page with book listing  
✔ User **registration & login (role-based access)**  
✔ Admin panel to **add / edit / delete books**  
✔ View book details  
✔ Built with:  
  - **Java Spring Boot**  
  - **Thymeleaf**  
  - **H2 In-Memory Database**  
  - **HTML & CSS** :contentReference[oaicite:1]{index=1}

## 🛠️ Installation

1. **Clone the repository**
   ```
   git clone https://github.com/Rahul-web-2/onlinebookstore.git

2. Navigate into the project directory

cd onlinebookstore

3. Build & Run

Using Maven:

mvn clean install
mvn spring-boot:run

Or run the project directly from your IDE (e.g., IntelliJ / Eclipse)

4. Access the app

http://localhost:8080
